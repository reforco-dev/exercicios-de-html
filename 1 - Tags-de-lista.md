## Listas 

Existem dois tipos de listas no HTML, as ordenadas e as não ordenadas.

### Ordenadas

```html
<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
    <li>Item 4</li>
    <li>Item 5</li>
</ol>
```

### Não Ordenadas

```html
<ul>
    <li>Item</li>
    <li>Item</li>
    <li>Item</li>
    <li>Item</li>
    <li>Item</li>
</ul>
```
